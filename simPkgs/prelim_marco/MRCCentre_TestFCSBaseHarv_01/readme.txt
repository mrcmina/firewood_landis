Description of contents and instructions
---

This package contains the input files for running a test simulation with ForCS and Base-Harvest for the landscape MRC Centre (Quebec).

There are two input file for Base-Harvest:
 - base_harvest_1N --> adaptation for Base-Harvest of prescriptions initially prepared for Biomass-Harvest (which is NOT compatible with ForCS)
 - base_harvest_random_clearcut --> a dummy Base-Harvest input file to be used for testing purposes
